# 📊 SQL Business Analysis & Reporting Project

## 📌 Project Overview

This project demonstrates a comprehensive business analysis using SQL on a Data Warehouse structured with a Gold Layer (Fact & Dimension tables).

The objective is to transform transactional data into meaningful business insights using advanced SQL analytical techniques.

---

## 🏗️ Data Architecture

The project is built on a Star Schema model:

- gold.fact_sales
- gold.dim_customers
- gold.dim_products

This structure simulates a real-world Data Warehouse (Bronze → Silver → Gold).
